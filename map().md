Функция map() в Python используется для обработки и преобразования всех элементов итерируемого объекта (например, списка, кортежа или строки) с помощью заданной функции. 
Она возвращает итератор, который можно преобразовать в список, кортеж или другой итерируемый объект.

`map(function, iterable, ...)`

- function: Функция, которая будет применяться к каждому элементу итерируемого объекта. Это может быть как lambda-функция, так и обычная именованная функция.
- iterable: Один или несколько итерируемых объектов (например, списков, кортежей, строк и т.д.).

## Применение функции к каждому элементу списка
  
    # Пример использования map() с lambda-функцией
    numbers = [1, 2, 3, 4, 5]
    doubled_numbers = map(lambda x: x * 2, numbers)
    print(list(doubled_numbers))  # Выведет: [2, 4, 6, 8, 10]

## Использование обычной функции с map()

    def square(x):
        return x ** 2
    
    numbers = [1, 2, 3, 4, 5]
    squared_numbers = map(square, numbers)
    print(list(squared_numbers))  # Выведет: [1, 4, 9, 16, 25]

## Применение функции к нескольким итерируемым объектам

Если передать несколько итерируемых объектов, map() будет применять функцию параллельно к элементам всех итерируемых объектов, пока самый короткий из них не закончится.

    numbers1 = [1, 2, 3, 4, 5]
    numbers2 = [10, 20, 30, 40, 50]
    
    summed_numbers = map(lambda x, y: x + y, numbers1, numbers2)
    print(list(summed_numbers))  # Выведет: [11, 22, 33, 44, 55]

## Преобразование строки в список символов ASCII-кодов

    characters = "hello"
    ascii_codes = map(ord, characters)
    print(list(ascii_codes))  # Выведет: [104, 101, 108, 108, 111]

## Применение функции к списку словарей

    people = [
        {'name': 'Alice', 'age': 25},
        {'name': 'Bob', 'age': 30},
        {'name': 'Charlie', 'age': 35}
    ]
    
    names = map(lambda person: person['name'], people)
    print(list(names))  # Выведет: ['Alice', 'Bob', 'Charlie']

Примечания

- map() возвращает итератор, что означает, что вы можете использовать его напрямую в циклах for, или преобразовать в другие коллекции, такие как списки или кортежи.
- Если функция не имеет возвращаемого значения (например, просто print()), map() вернет итератор с элементами None.
- Использование map() полезно, когда нужно применить одну и ту же операцию ко всем элементам итерируемого объекта, так как это позволяет писать более компактный и читаемый код.
